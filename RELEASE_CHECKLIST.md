# Release Checklist

## Build
- [ ] Version updated in title and visible tiny label.
- [ ] Service-worker cache name updated.
- [ ] Core rules self-tests pass in the browser console.
- [ ] ZIP contains no enclosing duplicate folder.

## iPhone smoke test
- [ ] New game and roll work.
- [ ] Legal checker/source highlighting works.
- [ ] Undo restores only the user's current turn.
- [ ] Computer starts only after the legal human sequence is complete.
- [ ] Rank my moves shows only legal complete sequences.
- [ ] Hint highlighting shows both source and destination.
- [ ] How to Play opens and closes.
- [ ] Tiny version label is visible without consuming useful space.

## Publish
- [ ] Upload extracted folder contents to repository root.
- [ ] Commit with versioned message.
- [ ] Confirm GitHub Pages deployment.
- [ ] Refresh Safari and reopen the Home Screen app while online.
